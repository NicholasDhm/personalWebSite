Dicionário de Tags HTML

Referência das principais tags HTML utilizadas neste portfólio.

Tag Descrição Exemplo
<!DOCTYPE> Define o tipo do documento e a versão do HTML. <!DOCTYPE html>
<html> Elemento raiz de um documento HTML. <html lang="pt-BR"></html>
<head> Contém metadados como título e links para estilos. <head></head>
<meta> Define metadados como codificação e viewport. <meta charset="UTF-8">
<title> Define o título exibido na aba do navegador. <title>Meu Site</title>
<link> Conecta a página a arquivos externos, como CSS ou ícones. <link rel="stylesheet" href="styles.css">
<body> Contém todo o conteúdo visível da página. <body></body>
<aside> Conteúdo lateral, como uma barra com perfil. <aside>Barra lateral</aside>
<img> Exibe uma imagem na página. <img src="imagem.jpg" alt="Descrição">
<div> Container genérico para agrupar conteúdo. <div>Conteúdo</div>
<h1> até <h3> Títulos e subtítulos, com diferentes níveis de importância. <h1>Título Principal</h1>
<p> Parágrafos de texto. <p>Este é um parágrafo.</p>
<a> Cria links para outras páginas ou seções. <a href="pagina.html">Clique aqui</a>
<nav> Define uma área de navegação com links. <nav><a href="#home">Home</a></nav>
<main> Conteúdo principal da página. <main>Conteúdo principal</main>
<section> Agrupa conteúdo relacionado. <section>Seção de conteúdo</section>
<ul> Lista não ordenada. <ul><li>Item 1</li></ul>
<li> Item dentro de uma lista. <li>Item de lista</li>
<span> Container inline usado para estilizar parte do texto. <span style="color: red;">Texto em vermelho</span>
<footer> Rodapé da página com informações de contato. <footer>Rodapé</footer>
<address> Informações de contato. <address>Endereço aqui</address>
<script> Contém ou referencia código JavaScript. <script src="script.js"></script>
<i> Ícones ou texto em itálico (aqui usado com Font Awesome). <i class="fa fa-home"></i>
